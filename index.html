¿Qué es el Código Morse?

El código Morse es un sistema de codificación que representa letras, números y signos de puntuación mediante una serie de señales cortas y largas: puntos (·) y rayas (-). Fue creado en el siglo XIX por Samuel Morse y Alfred Vail para transmitir mensajes a través del telégrafo.

¿Cómo usar el traductor?

Escribe texto normal o código Morse en el campo correspondiente, selecciona el modo de traducción, y el sistema convertirá automáticamente el contenido. Puedes traducir frases como "Hola Mundo" o códigos como ".... --- .-.. .- / -- ..- -. -.. ---".
